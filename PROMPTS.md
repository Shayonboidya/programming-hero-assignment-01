# Prompt (Conference Schedule Section)
```I am building a conference landing page called "DevConf 2026".

The website already has these sections:
1. Hero Section
2. Meet the Speakers
3. Pricing Section

I need a brand-new section placed after the Pricing section.

Create a modern and professional "Conference Schedule" section using ONLY HTML5 and CSS3.

Requirements:
- Use semantic HTML5 tags.
- Use ONLY HTML5 and CSS3.
- Do NOT use Bootstrap, Tailwind CSS, JavaScript, jQuery, React, Vue, or any other framework or library.
- Do NOT use inline CSS.
- Write separate HTML and CSS code.
- The layout must be fully responsive using CSS Flexbox and CSS Grid only.
- Maximum container width: 1200px.
- White background with blue accent colors that match a modern tech conference website.
- Include a section title and subtitle.
- Show a two-day conference schedule.
- Each schedule item should display:
  - Time
  - Session Title
  - Speaker Name
  - Session Type (Workshop, Keynote, Panel, etc.)
- Use modern cards with border-radius, subtle box shadows, spacing, and smooth hover effects.
- Add small circular timeline indicators or modern design elements using only CSS.
- Typography should look clean and professional.
- The section must visually match the Hero, Speakers, and Pricing sections.
- Write clean, well-organized, and commented code.
- Return ONLY the HTML and CSS code.```
